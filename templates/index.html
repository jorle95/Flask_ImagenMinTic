Reto práctico:

Desarrollar una aplicación Web utilizando HTML, CSS, Javascript y librerías y frameworks que necesite, que cumpla con las siguientes funciones:

  • Que permita el cargue de una imagen (formato jpg, jpeg y/o png)
  • Una vez cargada, debe permitir su visualización
  • Debe listar las características generales de la imagen (Ej: Resolución, formato, tamaño en Kb)
  • Al hacer o dar clic sobre algún punto de la imagen visualizada en la aplicación Web, debe presentar la siguiente información:
    • Coordenas del pixel seleccionado
    • Color en formato Hexagesimal (EJ. #62ACB7)
    • Color en formato RGB (EJ. 98, 172, 183)
